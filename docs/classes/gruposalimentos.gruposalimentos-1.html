Class GruposAlimentos

Clase abstracta para definir los grupos de alimentos. Se utilizará como recurso para completar el grupo alimenticio al que pertenece un ingrediente. Se trata de una clase abstracta, por lo que no se podrán realizar instancias de la clase.
